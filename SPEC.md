## 1. ゲーム概要

### ジャンル

**Gem Sort Puzzle（ジェムソートパズル）**  
カラーソートパズルジャンルの亜種。「カラーソート × ピクセルアート完成」が特徴。  
参考タイトル：[Gems Flow: Color Puzzle](https://apps.apple.com/us/app/gems-flow-color-puzzle/id6758003426)（WONDERPOP PTE. LTD）

### コアコンセプト

> グリッド上の混在したジェムを、同色グループごとに正しいエリアへ移動させよう。  
> 制限時間内にピクセルアートを完成させる達成感が核心体験。

- **時間制限あり**（ステージごとにタイマー：数分〜10分程度）
- 同色グループをまとめてタップ → 移動する操作感
- ステージクリアでピクセルアートが完成する演出
- ビジュアルテーマ：ファンタジー風（Synty 3D背景素材を活用予定）

## 2. コアゲームメカニクス（確定）

### 2.1 グリッド

- ゲーム画面は以下の2つの独立したグリッド領域で構成される：
  1. **ピクセルアートグリッド（メイン領域）**：横24マス×縦24〜36マスの矩形グリッド。各マスに「完成時の目標色（Goal Color）」が設定されている。
  2. **一時退避用パレット（パレット領域）**：画面下部にある **横16マス×縦4マス** の矩形グリッド。ジェムを一時的に退避させるための空きスペース。目標色の設定はない。
- 各マスにジェムが配置されているか空の状態。
- ジェムには**色**がある（赤・青・緑・黄・紫 など）。

### 2.2 操作ルール（確定）

| 操作 | 内容 |
|------|------|
| タップ（グループ選択） | 同色ジェムグループ全体を選択（ハイライト） |
| タップ（移動先指定） | 移動先の空きエリアを指定 → グループ全体が移動 |

**移動できる条件：**
1. 移動先に**グループ全体が収まる連続した空きマス**がある
2. または移動先が**同色グループの隣接エリア**（同色グループ同士の統合）

> グループ単位で動くため、「空間をどう確保するか」が戦略の核心。

### 2.3 クリア条件

- 全ジェムが**正しい色エリアに揃った状態**でクリア
- **制限時間内**にクリアが必要
- タイムオーバーでステージ失敗（リトライ可）

### 2.4 時間制限システム

- ステージごとに制限時間を設定（初期：5〜10分、後半：3〜5分）
- 残り時間はプログレスバーまたはカウントダウンで表示
- 時間切れ → 失敗画面 → リトライ / 広告視聴で時間追加 / ステージ選択へ

### 2.5 補助機能

| 機能 | 内容 |
|------|------|
| Undo | 1手前に戻る（回数制限なし、またはステージ内3回など要調整） |
| Reset | ステージ初期状態に戻す（タイマーもリセット） |
| ヒント | 次に動かすべきグループを示す（広告視聴 or ヒントポイント消費） |
| 時間追加 | 残り時間を+1〜2分延長（広告視聴 or アイテム消費） |

## 3. ステージ設計

### 3.1 基本パラメータ

| パラメータ | Easy | Normal | Hard |
|-----------|------|--------|------|
| グリッドサイズ | 横24×縦24 | 横24×縦30 | 横24×縦36 |
| パレットサイズ | 横16×縦4 (固定) | 横16×縦4 (固定) | 横16×縦4 (固定) |
| 色の種類 | 3〜4色 | 4〜6色 | 6〜8色 |
| 空きマス（メイン） | 多め | 中程度 | 少なめ |
| 制限時間 | 8〜10分 | 5〜7分 | 3〜5分 |
| 特殊マス | なし | なし | あり |

### 3.2 特殊マス（後半実装予定）

| 種類 | 説明 |
|------|------|
| ロックマス | 特定条件を満たすまで移動不可 |
| 固定ジェム | 動かせないジェム（障害物） |
| 隠し色 | 最初はグレー表示、移動すると色が判明 |

## 4. ピクセルアート演出

### 4.1 完成システム

- グリッドの各マスが完成絵の1ピクセルに対応
- ジェムが正しい位置に置かれると**そのマスのピクセルが点灯**
- 全マス確定 → クリアアニメーション → 完成したピクセルアートを表示

### 4.2 アートテーマ（初期予定）

- ファンタジー（魔法・ドラゴン・宝石モチーフ）
- 動物
- 風景
- 宇宙・銀河

## 5. ビジュアル / 演出方針

### 5.1 全体テイスト

- **ゲーム画面**：ピクセルアート調のジェムとグリッド
- **トップ画面・背景**：Synty Studios の 3D ファンタジー素材を使用
  - 例：`Polygon Fantasy Kingdom`、`Polygon Dungeons` など
  - カメラは固定アングル or ゆっくりパン
  - ライトエフェクト・パーティクルで豪華な雰囲気を演出

### 5.2 UIデザイン

- ゲームプレイ中はシンプルで見やすいUI
- ジェム選択時：浮き上がり + グロー演出
- グループ移動時：スライドアニメーション
- 色グループ完成時：スパーク・光の柱エフェクト
